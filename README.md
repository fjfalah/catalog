# catalog

Simple Catalog Interface inspired by Sorabel Catalog Interface.

### Feature
- Catalog List
- Catalog Detail
- Filter by Price / Color / Size
- Sort by Price / Most Popular / Latest

### Main Dependencies
- React Js (View)
- Rx Js (Middleware)
- Styled Components (CSS)
- Firestore by Firebase (Sample Database)
- Webpack (Bundler)

### How to Run

```bash
# development mode
npm install
npm run dev
```

```bash
# production mode
npm install
npm start
```
